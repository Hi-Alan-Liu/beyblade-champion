# 陀螺配置卡片產生器（Beyblade Ranking Card Generator）

純前端、無後端、無資料庫的工具，快速產生比賽名次卡並下載成 PNG。
靈感來自 HANIK TRACK 的賽事名次卡（1ST~4TH PLACE）。

![範例](範例僅供參考，實際輸出由你的內容組成)

---

## 功能總覽

- **4 種名次**：1ST(紅) / 2ND(藍) / 3RD(綠) / 4TH(紫)，切換即同步整張卡的主色
- **3 種匯出尺寸**：IG 4:5(1080×1350) / 限動 9:16(1080×1920) / 方形 1:1(1080×1080)，輸出 2x 高解析
- **每張卡 3 顆陀螺 × 3 零件**：上蓋 / 固鎖 / 軸心，名稱自動合併（例：上蓋「隕星龍騎士」+ 固鎖「8-70」+ 軸心「L」→ `隕星龍騎士8-70L`）
- **圖片**：人物照、右側面板背景、9 個零件圖，皆可上傳或**拖放**；上傳會**自動縮放壓縮**（長邊 ≤1400px）
- **零件圖呈現**：完整 / 圓角方框 / 圓框
- **多卡批次**：上方分頁可新增 / 切換 / 改名 / 刪除 / 拖曳排序 / 複製，一鍵「匯出全部」
- **常用字典**：內建常見零件名，可自行增補（存在瀏覽器）
- **自動暫存**：所有內容存在瀏覽器 localStorage，重整不流失
- **離線可用**：匯出函式庫已打包在 `vendor/`，無網路也能用

---

## 快速開始

1. 直接開啟 `index.html` 即可使用。
2. 建議用本機伺服器開啟（消除瀏覽器對 `file://` 的小限制，匯出更乾淨）：

   ```bash
   # 任選一種，在專案資料夾執行後開 http://localhost:8000
   python -m http.server 8000
   # 或
   npx serve
   ```

---

## 操作流程

1. 選**名次**與**匯出尺寸**
2. 填 **LOGO 文字 / 暱稱 / 右下浮水印**
3. 上傳**人物照**、（選填）**面板背景**
4. 逐顆陀螺：填**上蓋 / 固鎖 / 軸心**名稱（會出現自動補全建議），上傳或拖入三個零件圖
5. 需要多張就按 **＋新增卡片**（分頁可拖曳排序、改名、複製）
6. 按 **⬇ 下載目前卡片 PNG** 或 **⬇⬇ 匯出全部卡片**

## 如何擴充（改 `app.js` 最上方常數即可）

- **名次/配色**：改 `RANKS`（key 對應下拉值，`accent` 主色、`line1/line2` 名次字）
- **匯出尺寸**：改 `SIZES`（寬固定 1080，`h` 為高度）
- **內建字典**：改 `DICT.blade / ratchet / bit`（使用者增補的另存在 localStorage `beyblade-dict-v1`）
- **零件圖外框**：`shapeSelect` 選項 + `styles.css` 的 `.card[data-shape="..."]`
- **壓縮上限**：改 `MAX_EDGE`（預設 1400）

## 技術說明

- 純靜態三檔，無建置步驟、無相依套件（除離線打包的 html-to-image）
- 卡片用 HTML/CSS 排版（`.card` 固定 1080 寬、高隨尺寸），預覽用 `transform: scale()` 縮放
- 匯出用 `html-to-image` 的 `toPng`，`pixelRatio: 2` 取得高解析；以 `filter` 略過空圖與編輯用提示框
- 圖片一律走 dataURL（離線可用、可被截圖），上傳時 canvas 縮放壓縮
- 狀態存 localStorage（`beyblade-card-v1` 卡片、`beyblade-dict-v1` 字典）

---

## 已知限制

- localStorage 容量約 5–10MB；極多卡 + 大量原圖仍可能超量（已用上傳壓縮大幅降低風險）
- 用 `file://` 直接開啟時，匯出 console 會有 `cssRules`/`fetch` 警告（不影響輸出）；用本機伺服器即無
- 中文字型靠系統字型（Windows: 微軟正黑體 / Mac: 蘋方），不同系統字體略有差異
